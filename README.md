# Taaza-Khabar-App
Taaza Khabar: A real-time Android news aggregator built with Firebase (Auth &amp; Database) and Retrofit. Features a sleek Material Design UI with CardView, CoordinatorLayout, Swipe-to-Refresh, and Glide for high performance. Stay updated with categorized global news in tech, sports, and business through this immersive, secure platform.
